# 学习助手技能

当用户提到以下内容时自动激活此技能：
- 整理笔记、课程笔记、课堂笔记
- 学习计划、复习计划、周计划、学期计划
- 复习、备考、考前冲刺、考试
- 课程名称（高数、线代、C语言、数据结构等）

## 核心能力

### 1. 笔记整理
- 遵循 [[CLAUDE.md]] 中定义的笔记规范
- 使用 `99-模板/课程笔记模板.md` 的格式
- 笔记保存到 `01-课程/课程名/笔记/` 下
- 自动添加 Frontmatter（title, course, week, date, tags, status）
- 关联知识点之间建立 `[[Wiki链接]]`

### 2. 学习计划
- 使用 `99-模板/学期计划模板.md` 或 `99-模板/周计划模板.md`
- 保存到 `02-学习计划/` 文件夹
- 计划包含具体可执行的任务清单
- 支持动态调整（标记完成/未完成）

### 3. 复习备考
- 使用 `99-模板/复习提纲模板.md`
- 生成复习提纲、重点归纳
- 可出题自测（选择题、填空题、编程题）
- 分析错题、定位薄弱点

### 4. 迭代更新
- 用户提出优化建议时，更新 [[CLAUDE.md]] 中的版本日志
- 用户需求变化时，调整文件夹结构或模板
