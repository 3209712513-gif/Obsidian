# 🧠 各题型解题技巧全攻略

> ⏰ **做题顺序建议**：写作 → 听力（固定）→ **仔细阅读→翻译→匹配→选词填空**
> 💡 选词填空分值最低（5%），没时间直接蒙！

---

# 1️⃣ 写作 Writing（15% · 30min）


## 🧱 万能框架

```
┌─────────────────────────────────┐
│  Paragraph 1：引出话题+表明观点  │
│  Paragraph 2：论据1+论据2+论据3  │
│  Paragraph 3：总结+升华/建议     │
└─────────────────────────────────┘
```

## 🎯 高分策略

| 步骤 | 内容 | 时间 |
|:----:|:----|:----:|
| 1 | **审题**：圈出题目关键词，确定题型+立场 | 2min |
| 2 | **列提纲**：3段分别写什么，关键词写下来 | 3min |
| 3 | **写正文**：套模板+填充内容 | 20min |
| 4 | **检查**：语法错误+拼写+单复数+时态 | 5min |

## 🔥 加分技巧

| 技巧 | 低级表达 → 高级替换 |
|-----|-------------------|
| 开头禁用"I think" | `I firmly believe that...` / `From my perspective...` |
| "more and more"太啰嗦 | `increasingly / growing number of` |
| "important"太弱 | `crucial / vital / essential / significant / paramount` |
| "many"太普通 | `a multitude of / a host of / numerous` |
| "because"重复太多 | `due to / owing to / given that / on account of` |
| "so"太随意 | `therefore / consequently / as a result / hence` |
| "for example"变一下 | `for instance / to illustrate / a case in point is` |
| 结尾升华 | `Only by doing A can we achieve B.` (倒装结尾，加分！) |

# 2️⃣ 听力 Listening（35% · 30min）

## 📊 题型分布

| 部分 | 题型 | 题量 | 建议策略 |
|:----:|:----|:---:|:--------:|
| Section A | 长对话 **2篇** | 8题 | 对话内容容易跟上 |
| Section B | 篇章 **2篇** | 7题 | 注意文章主旨 |
| **Section C** | 讲座/讲话 **3篇** | **10题** | **最难！分值最高，重点攻克** |

## 🎯 核心技巧

### ① 预读选项（播放前抢时间！）
- **Directions** 时间不要听！用来**快速扫读选项**
- 圈出选项中的**核心名词+动词+否定词**（not, rarely, hardly）
- 通过选项反推**问题可能问什么**

### ② 边听边记
| 信号词 | 后面跟着答案的概率⬆️ |
|--------|:------------------:|
| **But / However** | ⭐⭐⭐⭐⭐ |
| **Because / So / Therefore** | ⭐⭐⭐⭐⭐ |
| **In fact / Actually** | ⭐⭐⭐⭐ |
| **The most important** | ⭐⭐⭐⭐ |
| **First / Second / Finally** | ⭐⭐⭐ |
| **For example / Such as** | ⭐⭐⭐ |
| **In conclusion** | ⭐⭐⭐ |
| **It turns out that** | ⭐⭐⭐⭐ |

### ③ 听不懂怎么办？
- **听到什么选什么**（六级听力的正确答案通常是原文复现或同义替换）
- **不要纠结已经过去的题**，立即看下一题选项
- 实在没听到 → 选最长的那个选项（统计概率较高）

### ④ Section C 讲座专攻
- 讲座一般有**总分结构**，开头第一句就是主旨
- 注意**列举**（first, another, also, in addition）
- 注意**语气强调**（重读、重复的词往往是答案）

# 3️⃣ 阅读 Reading（35% · 40min）

## ⏰ 时间分配建议

```
┌───────────────────────────────────────┐
│  Section C 仔细阅读  2篇  10题 → 20min │  ← 先做！
│  Section B 段落匹配  1篇  10题 → 12min │
│  Section A 选词填空  1篇  10题 → 8min  │  ← 最后做！
└───────────────────────────────────────┘
```

## 🅰 仔细阅读（分值最高！20%）

### 解题流程
```
Step 1：读题干（不读选项！）→ 圈关键词
  ↓
Step 2：关键词定位原文 → 找到出题句
  ↓
Step 3：仔细读懂出题句 + 上下文1句
  ↓
Step 4：比对选项 → 排除干扰项
```

### 六大题型 & 对策

| 题型 | 特征词 | 对策 |
|:----:|:-------|:-----|
| **主旨题** | mainly discuss / best title | 读首段+各段首句，找反复出现的词 |
| **细节题** | According to... / because... | 关键词定位原文，找同义替换 |
| **推断题** | imply / infer / suggest | 答案不在原文直接说，要**推理一步**；但别过度推理！ |
| **语义题** | "..." refers to / closest in meaning | 看划线词所在句+上下句 |
| **观点题** | What's the author's attitude? | 找态度词：positive/negative/skeptical/neutral |
| **例证题** | The example is used to... | 例子本身不重要，**例子支持的论点**才是答案 |

### 干扰项套路（必看！）
| 类型 | 描述 | 识别方法 |
|:----:|:----|:--------|
| 偷换概念 | 原文A > 选项B | 严格比对名词和限定词 |
| 以偏概全 | 部分内容说成全部 | 看有没有 all/most/some 等程度词 |
| 无中生有 | 原文没提 | 找不到原文依据的直接排除 |
| 正反混淆 | 意思说反了 | 注意 not/exclude/except 等否定词 |
| 过度推理 | "可能"变成"一定" | 注意选项中的绝对词：always, never, must |

> ⚡ **秒杀技巧**：两个选项意思相反 → 答案必是其中一个！

## 🅱 段落匹配（10%）

### 解题流程
```
Step 1：读10个题干，圈出关键词（专有名词、数字、核心名词）
  ↓
Step 2：从 A 段开始扫读，找关键词匹配或同义替换
  ↓
Step 3：匹配上的标出来，不确定的先跳过
  ↓
Step 4：剩下的段落二次扫读，排除法
```

### 🎯 技巧
- **先读题干，不看原文**！10个题干读完大概知道文章讲什么
- **同义替换是核心**：原文 ≈ 题干，不会一模一样
- **数字、大写、专有名词**是最容易定位的
- 每个段落顶多用一次，排除法很好用
- 选过的段落做个标记 ✅

---

## 🅲 选词填空（5% — 分值最少，最后做！）

### 解题流程
```
Step 1：标出15个词的词性（n./v./adj./adv.）
  ↓
Step 2：读原文，看空格缺什么词性
  ↓
Step 3：从对应词性中选符合语义的
```

### 🎯 技巧
- **先定词性，再选意思** — 极大缩小范围
- 动词注意**时态**（-ed / -ing / 原形）
- 名词注意**单复数**
- 固定搭配很常见（如：take sth into account）
- **实在不会就蒙**，反正只有5%分值

# 4️⃣ 翻译 Translation（15% · 30min）

## 🎯 核心原则

| 原则 | 说明 |
|:----:|:------|
| **信** | 准确翻译关键信息，不要漏译 |
| **达** | 英文句子要通顺，符合英文语法 |
| **不追求雅** | 时间有限，保底就好 |

## 🔥 三步法

```
Step 1：拆句子
  中文长句 → 拆成2-3个短句
  每个短句 = 1个主谓结构
  ↓
Step 2：找主干
  每个短句找出：主语 + 谓语 + 宾语
  先把主干写出来！
  ↓
Step 3：加修饰
  形容词、副词、从句往上加
  用 and/but/which/that 连接
```

## 💡 救命技巧

| 情况 | 怎么办 |
|:----:|:--------|
| **单词不会写** | 用简单的同义词！"宏伟的"不会magnificent → 用great |
| **句子太复杂** | 拆成2-3个简单句，用and连接 |
| **主动态不会** | 换被动态（或反之） |
| **中国文化词** | 拼音+解释，如：饺子 jiaozi (Chinese dumpling) |
| **时间/朝代** | in the Qing Dynasty / in the 18th century |

## ⚡ 必备句式
- **越来越...** → an increasing number of / increasingly
- **被称为** → known as / referred to as
- **发挥着重要作用** → play a significant/vital role in
- **不仅...而且...** → not only... but also...（倒装：Not only does it...）
- **随着...的发展** → With the development of... / As ... develops

# ⚡ 终极考场策略

## ⏰ 时间分配表

| 部分 | 时间 | 注意 |
|:----:|:----:|:------|
| **写作** | 30min | 模板默写+填充，提前5min检查 |
| **听力** | 30min | 一边听一边涂答题卡！**不单独留涂卡时间** |
| **仔细阅读** | 20min | ⭐最值钱，先做！ |
| **翻译** | 20min | 先保证主干翻译对，再润色 |
| **段落匹配** | 15min | 题干关键词定位 |
| **选词填空** | 5min | 最后做，来不及直接蒙 |

## ⚡ 蒙题技巧（只用于实在不会时）
- 听力选项：**最长的选项概率略高**
- 阅读选项：**两个相反选项 → 答案在其中**
- 选词填空：**全蒙同一个字母**（期望正确率 ≈ 25%）
