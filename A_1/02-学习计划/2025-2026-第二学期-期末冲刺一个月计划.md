# Phase 1：六级冲刺 + 打基础
## 📅 第1周（6月1日 ~ 6月7日）

### 六级 · 每日必做（≈3h/天）
| 项目 | 内容 | 时长 |
|------|------|------|
| 🎧 听力 | 每日1套听力真题Section A+B | 40min |
| 📖 阅读 | 每日2篇仔细阅读 + 1篇长篇匹配 | 50min |
| ✍️ 写作 | 每日背1个模板+写1篇作文 | 40min |
| 📝 翻译 | 每日1篇段落翻译 | 20min |
| 📚 词汇 | 六级核心词50个/天（用App刷） | 20min |
| **周末** | 做1套完整六级真题（计时） | 2.5h |

### C语言 · 补地基（≈1.5h/天）
- **目标**：过一遍基础语法，重点攻 **指针 + 数组 + 函数**
- **方法**：看教材例题 → 遮住答案自己写 → 写不出来再看 → 第二天重写
- 每日任务：
  - [ ] Day1-2：三大结构（顺序/选择/循环）代码练习，每日5道编程题
  - [ ] Day3-4：**数组**（一维+二维）+ 字符串处理，每日5题
  - [ ] Day5-7：**函数**（传参、递归、变量作用域）+ **指针**入门

### 离散数学 · 跟进度（≈1h/天）
- **目标**：跟上第二章，为后续赶超打基础
- [ ] 继续学第二章 谓词逻辑
- [ ] 做课后习题，重点练 **谓词翻译 + 等价/蕴含式推理**

### 高数 · 保进度（≈0.5h/天）
- [ ] 曲线微分部分继续学，跟上课堂进度
- [ ] 核心掌握：方向导数与梯度、多元函数极值
