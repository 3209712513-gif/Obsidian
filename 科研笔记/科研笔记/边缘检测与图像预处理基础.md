# 边缘检测与图像预处理基础

## 一、Canny 边缘检测

Canny 算子包含五个核心步骤：

### 1. 灰度化
彩色图像转换为单通道灰度图，简化后续处理。

### 2. 高斯滤波
用高斯核对图像做卷积，抑制高频噪声。

**为什么必须先做高斯滤波？**
图像传感器、压缩噪声等会在像素值中引入随机扰动。Canny 的后续步骤（求梯度）对噪声极其敏感——一个随机噪点可以产生很大的局部梯度，被误判为边缘。高斯滤波的作用就是在不严重破坏真实边缘的前提下，先把这些扰动压下去。

### 3. 梯度计算（Sobel 算子）
用 Sobel 核分别计算水平方向梯度 Gx 和垂直方向梯度 Gy，得到梯度幅值和方向。

### 4. 非极大值抑制（NMS）
沿每个像素的梯度方向，与前后邻居比较，只保留局部最大值，将粗边缘细化为单像素宽度的边缘线。

### 5. 双阈值检测 + 边缘连接
设置高阈值 T_high 和低阈值 T_low：
- 幅值 > T_high -> 强边缘，直接保留
- T_low < 幅值 < T_high -> 弱边缘，仅当与强边缘相连时保留
- 幅值 < T_low -> 直接丢弃

### OpenCV 实现
```python
import cv2
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)
blurred = cv2.GaussianBlur(img, (5, 5), 1.4)
edges = cv2.Canny(blurred, threshold1=50, threshold2=150)
```

## 二、Sobel 算子详解
Sobel 算子是一对 3x3 卷积核，分别沿水平和垂直方向估算图像亮度的偏导数。边缘 = 亮度变化剧烈的地方，而导数正是度量变化快慢的工具。

## 三、卷积核（Kernel / Filter）
卷积核就是一个小矩阵，在图像上逐块滑动，对每个位置做加权求和，输出一个新值。核的内容不同效果就完全不同：模糊、锐化、边缘检测都是靠改变权重实现的。

## 四、灰度线性变换
把图像的灰度范围拉伸或压缩到想要的区间，让对比度更清晰。

## 五、中值滤波（Median Filter）
对滑动窗口内的各像素灰度值按升序排列，用其中值代替窗口中心像素的灰度值。

### 为什么比均值滤波好？
均值滤波取平均，异常值直接把平均数拉偏了。中值滤波取中位数，异常值天然被隔离。

```python
img_denoised = cv2.medianBlur(img, 5)
```

## 六、HSV 色彩空间
- H（Hue，色调）：颜色是什么，0~180
- S（Saturation，饱和度）：颜色有多鲜艳，0~255
- V（Value，亮度）：有多亮，0~255

HSV 更适合做颜色分割，可以设定 H、S、V 范围来筛选特定颜色的区域。
