# 📚 CV 吃水线检测 · 学习导航

## 🎯 学习路线（推荐顺序）

### 第一阶段：图像处理基础
- [ ] 灰度化、滤波（高斯/中值/均值）
- [ ] 边缘检测（Canny、Sobel、Laplacian）
- [ ] 形态学操作（腐蚀/膨胀/开闭运算）
- [ ] HSV 色彩空间与颜色分割
- [ ] 阈值处理（全局/自适应/Otsu）

### 第二阶段：传统 CV 方法
- [ ] 直线检测（Hough Transform）
- [ ] 轮廓检测与拟合
- [ ] 特征点匹配
- [ ] 图像透视变换

### 第三阶段：深度学习入门
- [ ] CNN 基础（卷积、池化、全连接）
- [ ] 语义分割（U-Net、FCN）
- [ ] 目标检测（YOLO、Faster R-CNN）
- [ ] 数据增强

### 第四阶段：吃水线检测专项
- [ ] 文献调研（国内外已有方法）
- [ ] 复现 baseline
- [ ] 改进创新
- [ ] 船体数据集构建
- [ ] 实船测试
